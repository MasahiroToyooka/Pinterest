# Pinterest

Pinterestのようにコンテンツの高さによってセルの高さを変えるアプリを作りました。
カスタムレイアウトクラスを作成して作っています。今後はcellをタップしたら、ズームインしながら画面遷移できるように実装します。

スライドメニューは以前作った物を再利用しています。
